# EDA Titanic Project

A beginner-friendly data science project focused on:
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering

## Dataset
Titanic dataset from Seaborn / Kaggle

## Tools Used
- Python 3
- Pandas, NumPy, Matplotlib, Seaborn
- Jupyter Notebook
- VS Code

## Project Structure
data/raw/         → original dataset (not tracked by git)
data/processed/   → cleaned data (not tracked by git)
notebooks/        → jupyter notebooks
outputs/          → saved plots and figures
